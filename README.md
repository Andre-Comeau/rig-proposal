# rig-proposal

Robotics Interest Group (RIG)

This repository contains a **visual, HTML-based proposal** for a proposed **Robotics Interest Group (RIG)** hosted under the Protocol Institute.

The RIG is conceived as a **lightweight, volunteer governance layer** for applied research in agentic robotics — a place where industry partners, researchers, and end users can collaboratively shape projects, without over-formalizing structure too early.

## What this is
- A visual working artifact, not a product site
- A governance and project-delivery concept
- Designed to be viewed as a static HTML page
- Optimized for clarity on desktop and mobile (including iOS)

## What this is not
- A finalized consortium agreement
- A legal entity or incorporated body
- An active funding application
- A software library or framework

## Contents
- `index.html` — main visual proposal
- `images/` — diagrams illustrating governance, project flow, and funding model

## How to view
For correct rendering (especially on iPhone / iPad), the page should be served over **HTTP/HTTPS**.

Recommended options:
- Netlify (drag-and-drop hosting)
- GitHub Pages
- A local HTTP server

Opening the HTML file directly via `file://` (e.g. from the iOS Files app) may result in missing images due to platform security restrictions.

## Status
This is an exploratory, early-stage artifact intended to support discussion, iteration, and collaboration.

Feedback, forks, and adaptations are welcome.
